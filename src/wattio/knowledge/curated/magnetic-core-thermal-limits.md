# Magnetic Core Thermal Limits

## Overview
Estimated maximum power dissipation per core, depending on cooling conditions. These are **approximate** reference values — use with appropriate margin. Values are based on a baseline ΔT = 40°C.

Source: TDK ferrites databook, Ferroxcube datasheets.

## Scaling Formulas
All values can be scaled for different temperature rises and cooling conditions:

```
Pmax_core = baseline_Pmax_core × (ΔT / 40) × (h / h_base)
Pmax_total = Pmax_core × total_multiplier
```

### Determining ΔT
1. If the user specifies max temp rise → use it directly
2. If the user specifies max temperature → ΔT = Tmax - Tambient
3. If neither is specified → assume Tmax = 110°C, then ΔT = 110 - Tambient

### Cooling Parameters

| Cooling Condition | h (W/m²·K) | Total Multiplier | Notes |
|---|---|---|---|
| Natural convection | 8 | 1.6 | Ratio total/core losses |
| Heatsink (one face) | 18 | 2.0 | Heatsink-coupled face equivalent |
| Forced air 2 m/s | 23.59 | 1.8 | Empirical air-speed correlation |
| Liquid cooling | 300 | 2.3 | Nominal equivalent value |

- **Pmax Core** = max power the core alone can dissipate
- **Pmax Total** = max total losses (core + winding) the assembly can dissipate
- **Total Multiplier** = approximate ratio of total losses / core losses that the assembly can handle
